#HAIKU/コトバを集めて、HAIKUを生み出す不思議なジェネレーター

初めて制作したオリジナルアプリ「HAIKU」です。ランダムな俳句を生成します。ユーザーが言葉を覚えさせていく仕様になっています。フレームワークはRuby on Rails、画像生成にはRmagic、サーバーはHerokuを使用しています。

#Dependency
Ruby 2.5.1 Ruby on Rails 5.2.1
